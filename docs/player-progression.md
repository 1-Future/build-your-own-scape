# Player Progression

How players advance from level 1 to endgame and beyond. XP curves, milestones, endgame systems, and what keeps players engaged long-term.

---

## XP Curve

| Property | Description |
|---|---|
| OSRS Formula | Exponential: level N needs floor(N + 300 * 2^(N/7)) / 4 cumulative XP |
| Linear Formula | Fixed XP per level, scales evenly |
| Custom Curve | Dungeon master defines arbitrary XP requirements per level |
| Total XP at 99 | ~13M (OSRS default) |
| Total XP at 120 | ~104M (RS3 default) |
| Virtual Levels | Levels beyond the cap, cosmetic only, toggle per world |
| Level Cap | Configurable per skill (default 99) |
| 200M XP Cap | Hiscore tracking limit, no further XP recorded beyond this point |

---

## Combat Level

| Property | Description |
|---|---|
| Input Skills | Attack, Strength, Defence, Hitpoints, Ranged, Magic, Prayer |
| Path Selection | Melee vs Ranged vs Magic -- highest determines combat style contribution |
| Max Combat Level | 126 (OSRS formula) |
| Used For | PvP matching, monster aggression rules, content gating |

---

## Total Level

Sum of all individual skill levels across the account.

| Milestone | Unlocks |
|---|---|
| 500 | Total level world access |
| 750 | Restricted chat channels |
| 1000 | Grand Exchange access (if gated) |
| 1250 | Additional restricted areas |
| 1500 | Total level world tier 2 |
| 1750 | Total level world tier 3 |
| 2000 | Total level world tier 4 |
| 2277 | Maximum total level (all 99s across 23 skills) |

## Milestone Rewards

### Skill Milestones

| Level | Reward |
|---|---|
| 50 | Mid-level unlock acknowledgment |
| 75 | High-level unlock acknowledgment |
| 92 | Halfway to 99 by XP (due to exponential curve) |
| 99 | Skillcape and emote |
| 120 | Master cape (RS3 style, toggle) |
| 200M | Virtual max, hiscore freeze |

### Account Milestones

| Cape | Requirement |
|---|---|
| Quest Cape | All quests completed |
| Achievement Cape | All achievement diaries completed |
| Music Cape | All music tracks unlocked |
| Max Cape | All skills at level cap |
| Completionist Cape | All content completed (quests, diaries, logs, music) |
| Custom Milestones | Dungeon master defines additional milestones and rewards |

---

## Endgame Systems

What players do after reaching max level or completing primary content.

| System | Description |
|---|---|
| Boss Hunting | Chase pets, rare drops, and kill count milestones |
| Collection Log | Complete the log by obtaining every unique item |
| PvP | Compete against other players in the wilderness or arenas |
| Combat Achievements | Tiered boss challenges with specific restrictions |
| Speedrunning | Complete quests or bosses in the fastest time possible |
| Economy | Flipping, merching, and market manipulation at the Grand Exchange |
| Clans | Wars, events, territory control, and clan progression |
| Helping New Players | Mentoring, guiding, and community building |
| Alternate Accounts | Fresh starts with different restrictions or goals |
| Game Modes | Ironman, hardcore, leagues, and custom modes |
| World Building | Player-built structures if the dungeon master allows it |

---

## Prestige / Rebirth

Reset a skill or entire account in exchange for prestige rank and cosmetic rewards.

| Property | Description |
|---|---|
| Reset Scope | Single skill or full account |
| Prestige Counter | Tracks number of resets (Prestige 1, 2, 3... unlimited) |
| Cosmetic Reward | Exclusive title, hiscore ranking, visual effects per tier |
| XP Bonus | Optional small XP bonus per prestige level (toggle) |
| Hiscore Ranking | Separate prestige hiscores |

---

## Seasonal Progression

| Property | Description |
|---|---|
| Mode | Separate fresh-start account with accelerated XP |
| Features | Relics, tasks, points, unique mechanics per season |
| Duration | Typically 6-8 weeks |
| Reward Transfer | Cosmetics and bank space transfer to main game on season end |
| Rankings | Season-specific hiscores and trophies (bronze, silver, gold, platinum) |

---

## Achievement Diary / Task System

| Property | Description |
|---|---|
| Structure | Region-based task lists |
| Tiers | Easy, Medium, Hard, Elite |
| Requirements | Specific skill levels, quest completions, activity completions |
| Reward Types | Items, buffs, area perks |
| Area Perks | Increased drop rates, free teleports, shop discounts |
