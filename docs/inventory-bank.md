# Inventory and Bank

## Inventory

| Feature | Description |
|---|---|
| Slot Count | Default 28, configurable |
| Weight Limit | Toggle -- only applies if weight system is enabled |
| Stack Rules | Define which items stack in a single slot |
| Inventory Layout | Configurable grid size |
| Drop Mechanics | Dropped items appear on the ground |
| Destroy Option | Items vanish permanently instead of dropping |
| Inventory Presets | Save and load predefined loadout configurations |
| Quick-Equip | Click an item in inventory to equip it directly |
| Shift-Click Drop | Toggle -- hold shift and click to drop items quickly |

## Bank

| Feature | Description |
|---|---|
| Total Capacity | Configurable base capacity with bonus from security features |
| Bank Tabs | Number of tabs is configurable |
| Tab Naming | Custom names for each bank tab |
| Placeholders | Empty slot remembers the item that was there (toggle) |
| Search Function | Search bank contents by item name |
| Bank Value Display | Show total value of bank contents |
| Item Sorting | Sort by name, value, type, or quantity |
| Bank Presets / Loadouts | Save and load gear + inventory configurations |
| Withdraw Modes | Withdraw as item, noted, or last used quantity |
| Deposit All | Button to deposit entire inventory at once |
| Bank PIN Protection | Require PIN entry to access bank |
| Separate Bank Per Mode | Toggle -- each game mode gets its own bank |
| Shared Bank | Shared bank storage for group ironman accounts |
| Bank Space Expansion | Additional space earned, purchased, or achievement-based |

## Additional Features (Plugin Audit)

| Feature | Description |
|---|---|
| Item Lock | Lock specific items to prevent withdrawal, sale, drop, trade, or alch. Locked items display a lock icon. Requires confirmation to unlock |
| Bank Changelog | Track all bank changes over time. Shows what was added, removed, and when. Daily diff view showing net changes |
| Bank Usage Heatmap | Visual heatmap showing most frequently accessed bank items. Helps identify what to keep accessible vs store deep |
| Multi-Term / Fuzzy Bank Search | Search bank with multiple terms simultaneously (AND/OR). Fuzzy matching so "addy" finds "adamant." Boolean operators supported |
| Bank Snapshot Sharing | Generate shareable image of bank contents with values. Share with friends or post externally. Privacy filter (hide specific tabs) |
| Banked XP Calculator | Calculate total XP stored in bank from processable items. Shows per-skill breakdown (herbs = herblore XP, ores = smithing XP). Tracks value over time |
| Unified Item Locator | Find where any item is across all storage: bank, inventory, equipment, group shared storage, death storage, house storage, looting bag. One search, all locations |
| Bank Data Export | Export bank contents to JSON/CSV for external tools, spreadsheets, or third-party applications. Scheduled or manual |
| Bank Audit / Junk Detector | Identify items taking up bank space with low/no value. Suggest items to sell, alch, or destroy. Configurable value threshold |
