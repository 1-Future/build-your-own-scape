# Tutorial and Onboarding

The new player experience -- from account creation to "I know how to play." The most critical 30 minutes of any MMO.

---

## Character Creation

Covered in asset-system.md (identity kit) and character-overview.md. Body type, parts, colors, and name selection. Quick presets for players who do not want to customize. Preview before confirming.

---

## Tutorial Island / Zone

Dedicated starting area separate from the main world. Step-by-step guided introduction.

| Step | Covers |
|---|---|
| Movement | Click to walk, run toggle |
| Camera | Camera rotation and zoom |
| Chat | Public, private, clan, and trade chat |
| Inventory | Picking up items, using items, dropping |
| Equipment | Equipping gear, stat changes, combat styles |
| Combat Basics | Attacking a monster, eating food, death (safe) |
| Gathering | One gathering skill introduction (e.g. fishing or mining) |
| Processing | One processing skill introduction (e.g. cooking or smithing) |
| Banking | Depositing and withdrawing items, bank organization |
| Map / Minimap | World map, minimap icons, navigation |
| Settings | Graphics, audio, keybinds, accessibility |

| Property | Description |
|---|---|
| NPC Guides | Scripted dialogue, not AI-generated |
| Progress Gating | Cannot leave until all steps are completed |
| Skip Option | Toggle -- dungeon master decides whether experienced players can skip |

## Guided Early Game

What happens after the tutorial zone.

| Feature | Description |
|---|---|
| Breadcrumb Quests | Short quests that lead the player to the first major town |
| Direction NPC | "Talk to the Duke" style -- gives direction without forcing a path |
| Map Markers | Key NPCs highlighted on the map (banker, shop, quest giver) |
| First Quest | Designed specifically to teach quest mechanics |
| Achievement Popups | Celebrate first milestones (first 10 levels, first item crafted) |

## Help System

| Feature | Description |
|---|---|
| Help Panel | In-game searchable FAQ |
| Advisor NPC | Context-sensitive tips based on what the player is currently doing |
| Hint Arrows | Point to relevant NPCs and objects during guided content |
| Task List | Suggested next actions for new players ("try fishing", "visit the mining guild") |
| Wiki Integration | Click any item or NPC to look up on the wiki |
| Loading Screen Tips | Rotating tips displayed during load screens |
| Opt-Out | Experienced players can disable all hints and suggestions |

---

## Mentor System

| Feature | Description |
|---|---|
| Mentor Status | Experienced players apply and are approved as mentors |
| Mentor Perks | Special chat color, mentor channel access, XP bonus when helping near new players |
| Request Assistance | New players can request help from an available mentor |
| Mentor Matching | Auto-pair a new player with an available mentor |
| Mentor Rewards | Cosmetics and titles for reaching mentoring milestones |

---

## Difficulty / Pacing

| Phase | Difficulty |
|---|---|
| Tutorial | Impossible to die, forgiving mechanics |
| Early Game | Gradual ramp, safe deaths, low penalties |
| Mid Game | Standard difficulty |
| Late Game | Full mechanics, no safety nets |

| Property | Description |
|---|---|
| Global Adjustment | Dungeon master can adjust the difficulty curve server-wide |
| Adaptive Hints | Show more tips if the player seems stuck, fewer if they are progressing smoothly |

## Returning Players

| Feature | Description |
|---|---|
| Welcome Back Flow | Triggered for players who have not logged in for a configurable number of days |
| Change Summary | Overview of what has changed since the player's last login |
| Gear Check | Notification if equipped gear is outdated or inappropriate for current content |
| Quick-Start Suggestions | Recommendations based on where the player left off |
