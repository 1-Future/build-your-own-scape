# Death System

## Death Mechanics

| Type | Description |
|---|---|
| Safe Death | Keep all items, respawn at designated location (minigames, certain bosses) |
| Dangerous Death | Lose items based on configured rules |
| Hardcore Death | Permanent consequences -- mode conversion or account deletion |

## Item Loss Rules

| Rule | Description |
|---|---|
| Items Kept on Death | Default 3, or 4 with Protect Item prayer |
| Untradeable Behavior | Kept, broken and needs repair, or lost entirely |
| Item Value Calculation | GE price, alch value, or custom value |
| Skulled Death | Keep 0 items |
| Item Protection Priority | Highest value items kept first |

## Gravestone System

| Feature | Description |
|---|---|
| Grave Spawn | Grave appears at exact death location |
| Grave Duration | Configurable timer (default 15 minutes) |
| Grave Visibility | Toggle -- whether other players can see and loot your grave |
| Reclaim From Grave | Free or GP cost to reclaim items |
| Grave Collapse | Items appear to everyone on collapse, then despawn after a period |
| Death Office | NPC holds items if grave is unreachable; fee based on item value |

## Death Cost

| Feature | Description |
|---|---|
| Free Deaths | Certain content has no death cost |
| Flat Fee Deaths | Fixed GP cost on death |
| Value-Based Fee | Percentage of total item value |
| Fee Cap | Maximum possible death cost |
| Fee Destination | Gold sink (removed from economy) |
| Free Death Week | Grace period for new content releases |

## Respawn

| Feature | Description |
|---|---|
| Respawn Location | Fixed spawn, last checkpoint, home teleport, or custom |
| Respawn Delay | Instant or configurable timer |
| Stat Restoration | Full or partial stat restoration on respawn |
| Run to Grave | Items stay at death location; player races back to reclaim |

## PvP Death

| Feature | Description |
|---|---|
| Killer Receives Loot | Configurable rules for what the killer gets |
| Skull Determines Drops | Skull status controls what the victim loses |
| Loot Keys | PvP loot stored in a key item, opened at a bank |
| Smite Mechanic | Drain opponent prayer to force them to drop an additional item |
| PvP-Specific Drop Rules | Separate drop rules from PvE deaths |
