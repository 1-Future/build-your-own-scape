# Friends and Social

## Friends List

| Feature | Description |
|---|---|
| Add/Remove | Add or remove players from friends list |
| Favorite | Mark specific friends as favorites or best friends |
| Notes | Attach personal notes to each friend |
| Friend Capacity | Maximum friend list size, determined by DM configuration |
| Best Friends / Favorites | Prioritized notifications for selected friends |
| Friend Groups / Tags | Organize friends by category (PvP friends, skilling friends, IRL friends) |
| Friend Activity Feed | Toggle-based feed showing what friends are currently doing |

## Ignore List

| Feature | Description |
|---|---|
| Blocked Players | Maintain a list of blocked players |
| Block Scope | Blocked from messaging, trading, and following |

## Social Features

| Feature | Description |
|---|---|
| Referral System | Invite new players; both referrer and referred receive rewards |
| Player Profiles | Public-facing stats and information |
| Player Search | Find players by name, total level, or clan |
| Nearby Players List | See players in your immediate area |
| Recent Interactions List | Track players you have recently interacted with |

## Module Toggles

| Toggle | Default |
|---|---|
| Friend Groups | Off |
| Activity Feed | Off |
| Referral System | Off |
| Player Profiles | Off |
| Player Search | Off |

## Additional Features (Plugin Audit)

| Feature | Description |
|---|---|
| Player Presence Status | Rich presence system: online, away, busy, invisible, do not disturb, in combat, in raid, AFK. Auto-status based on activity. Custom status message. |
| Live Friend Location Map | Real-time map showing friends' current in-game locations. Toggle visibility per friend. Click to teleport near (if allowed). |
| Friends List Export/Import | Export friends list to file and import on another account/device. Backup and restore. Merge without duplicates. |
| Item Lending Tracker | Track items lent to and borrowed from friends. Auto-return timer, lending history, and outstanding loans dashboard. |
| LFG / Group Finder | Looking-for-group matchmaking board. Post what activity you want to do, required levels, group size. Browse and join posted groups. Filter by activity, level, and time. |
| Name Change History | Track when friends change their display names. Shows previous names with dates. Helps identify renamed friends. |
| Spectate / Screen Share | Watch a friend play in real-time through their viewport. Spectator mode with optional commentary via voice chat. Useful for coaching and entertainment. |
| Timezone Display | Show friends' local timezones for coordinating play sessions. Auto-detected or manually set. Shows current time per friend. |
| Shareable Ban List | Export and import ban/ignore lists between clan admins. Community-maintained block lists for known scammers/bots. |
