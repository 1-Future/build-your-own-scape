# Friends and Social

## Friends List

| Feature | Description |
|---|---|
| Add/Remove | Add or remove players from friends list |
| Favorite | Mark specific friends as favorites or best friends |
| Notes | Attach personal notes to each friend |
| Friend Capacity | Maximum friend list size, determined by DM configuration |
| Best Friends / Favorites | Prioritized notifications for selected friends |
| Friend Groups / Tags | Organize friends by category (PvP friends, skilling friends, IRL friends) |
| Friend Activity Feed | Toggle-based feed showing what friends are currently doing |

## Ignore List

| Feature | Description |
|---|---|
| Blocked Players | Maintain a list of blocked players |
| Block Scope | Blocked from messaging, trading, and following |

## Social Features

| Feature | Description |
|---|---|
| Referral System | Invite new players; both referrer and referred receive rewards |
| Player Profiles | Public-facing stats and information |
| Player Search | Find players by name, total level, or clan |
| Nearby Players List | See players in your immediate area |
| Recent Interactions List | Track players you have recently interacted with |

## Module Toggles

| Toggle | Default |
|---|---|
| Friend Groups | Off |
| Activity Feed | Off |
| Referral System | Off |
| Player Profiles | Off |
| Player Search | Off |
